    부록 B: 환경 변수 참조¶

    public 신뢰도: high

    공개 SDK 기준 재작성¶

    공개판의 환경 변수는 이 저장소의 .env.example, Agent SDK docs, provider docs에서 확인 가능한 값만 다룬다. API key, provider selection, GitHub token, hosting env, managed agents credentials는 각 cookbook README와 .env.example에 연결한다.

    공개 경계¶

    • 실제 secret 값은 절대 게시하지 않는다.
    • 비공개 env var 추정 목록은 제외한다.

    실습 방향¶

    • 각 notebook 실행 전 필요한 env var를 README 수준에서 확인한다.
